# FoodFirestoreMVVM
This is a clean architecture app example built with

- Coroutines
- MVVM architecture
- Dagger Hilt
- Retrofit
- FirebaseFirestore
- Coroutines Flow
- ViewBinding

 Firestore provides offline storage by default.No need for room db.
